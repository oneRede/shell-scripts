# 台湾经济数据监控

收集台湾的主要经济指标，包括 GDP 增长率、出口额、进口额。

## 数据指标

- **GDP增长率**：年增率（%），季度数据
- **出口额**：月度数据，单位：亿美元
- **进口额**：月度数据，单位：亿美元
- **贸易顺差**：自动计算（出口-进口）

## 数据来源

本脚本采用**混合模式**：GDP 自动获取 + 进出口手动更新

1. **台湾主计总处**（GDP数据）✅ 自动获取
   - 网址：https://www.stat.gov.tw/Point.aspx?sid=t.1&n=3580&sms=11480
   - 方式：自动从网页提取最新数据
   - 频率：每周一自动运行

2. **财政部关务署**（进出口数据）⚠️ 需手动更新
   - 网址：https://portal.sw.nat.gov.tw/APGA/GA29
   - 方式：需下载Excel后手动输入
   - 原因：数据在Excel文件中，无直接API

3. **中央银行**（备用数据源）
   - 网址：https://www.cbc.gov.tw/

## 使用方法

### 1. 手动更新数据

每周运行一次，从官方网站查询最新数据后手动输入：

```bash
cd ~/git/sh/tw-economy
./update_data.sh
```

根据提示输入：
- GDP增长率（%）和数据期间（如：2024-Q2）
- 出口额（百万美元）和月份（如：2024-06）
- 进口额（百万美元）和月份（如：2024-06）

### 2. 记录数据

手动更新后，运行主脚本记录数据：

```bash
./get_tw_economy.sh
```

### 3. 自动运行（已配置）

脚本已设置为每周一上午 9:00 自动运行，将记录缓存中的数据。

1. **每周一上午**：
   - 访问官方统计网站查询最新数据

## 数据说明

### GDP增长率
- **单位**：百分比（%）
- **类型**：实质GDP年增率
- **频率**：季度发布
- **发布时间**：每季度末次月底

### 出口额
- **单位**：亿美元
- **频率**：月度发布
- **发布时间**：次月10日左右
- **涵盖**：所有商品出口总额

### 进口额
- **单位**：亿美元
- **频率**：月度发布
- **发布时间**：次月10日左右
- **涵盖**：所有商品进口总额

### 贸易顺差/逆差
- **单位**：亿美元
- **计算**：出口额 - 进口额
- **正值**：贸易顺差
- **负值**：贸易逆差

## 注意事项

1. 数据需要手动更新，建议设置日历提醒
2. GDP数据为季度数据，更新频率较低
3. 出口进口数据为月度数据，更新较频繁
4. 缓存文件（.cache）不要手动修改
5. 如需批量历史数据，可从官网下载后导入CSV
